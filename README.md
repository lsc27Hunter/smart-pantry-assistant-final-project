# Smart Pantry & Nutrition Assistant

A full-stack web application that helps users reduce food waste and discover recipes using ingredients already available in their pantry.

---

## Features

- Pantry ingredient tracking
- Recipe suggestions using Spoonacular API
- Pantry match percentage calculations
- Weekly meal planning
- Nutrition-aware recipe recommendations
- LocalStorage persistence
- Responsive mobile-friendly UI

---

## Technologies Used

### Frontend

- HTML5
- CSS3
- JavaScript (ES6)

### Backend

- Node.js
- Express.js

### External API

- Spoonacular API

### Other Tools

- Docker
- Excalidraw (UI prototype)

---

## Installation

### Clone repository

```bash
git clone YOUR_GITHUB_REPO_URL
```

### Install dependencies

```bash
cd backend
npm install
```

### Run backend server

```bash
npm start
```

### Run frontend locally

```bash
cd frontend
python3 -m http.server 5500
```

Open:

```text
http://localhost:5500
```
